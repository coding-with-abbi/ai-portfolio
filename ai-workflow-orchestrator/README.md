# AI Workflow Orchestrator

A modular system for orchestrating AI agents to plan and execute tasks using various tools.

## Structure

- `app.py`: Entry point
- `agents/`: Agent implementations (Planner, Executor)
- `tools/`: Tool implementations (Web Search, Python Runner, File Writer)
- `workflows/`: Workflow definitions and runner
- `memory/`: State management
- `config/`: Configuration
- `prompts/`: System prompts
